Moz Metrics

This tool enables you to find the Moz metrics of websites. You'll need to use a Moz token and access ID to use this system. If you don't have a token and access ID, you can request them (for free) from Moz.

Domain Metrics

Checks Moz metrics, such as Domain Authority and Page Authority, of the provided domain name.
